Hero Randomizer API

A quick guide to using the Hero Randomizer API

Introduction

Welcome to the Hero Randomizer API! This API allows you to generate random heroes or create them from predefined blueprints. Below are the available endpoints and examples to get you started.

Endpoints

GET /api/hero

Generates a completely random hero.

Try it

GET /api/hero?blueprint=...

Generates a hero based on the provided blueprint.
